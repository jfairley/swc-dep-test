# swc-dep-test

For Artifactory support, please use this sample project to debug the Unauthorized access errors for .swc files.

## Maven

Using Maven 3.3.3

```
mvn package -X
```

See logs: [maven.log](maven.log)


## Gradle

Using gradle 2.6

```
gradle build --debug
```

See logs: [gradle.log](gradle.log)
